# Purchase Receipts

Purchase receipts are read-only records created when goods are received from vendors. They are generated by posting purchase orders (receive action). Receipts cannot be created or modified via the API.

---

## Base Endpoint

```
{{BC_BASE_URL}}/companies(name='{{BC_COMPANY_NAME}}')/purchaseReceipts
```

---

## Operations

### 1. List All Purchase Receipts

```http
GET {{BC_BASE_URL}}/companies(name='{{BC_COMPANY_NAME}}')/purchaseReceipts
Authorization: Bearer {access_token}
Accept: application/json
```

**Response:**

```json
{
  "value": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "number": "RCPT-1001",
      "orderNumber": "PO-1001",
      "vendorNumber": "10000",
      "vendorName": "Fabrikam, Inc.",
      "postingDate": "2024-06-15",
      "lastModifiedDateTime": "2024-06-15T14:00:00Z"
    }
  ]
}
```

### 2. Get Receipt Lines

```http
GET {{BC_BASE_URL}}/companies(name='{{BC_COMPANY_NAME}}')/purchaseReceipts({prId})/purchaseReceiptLines
Authorization: Bearer {access_token}
Accept: application/json
```

**Response:**

```json
{
  "value": [
    {
      "id": "b2c3d4e5-f6a7-8901-bcde-f23456789012",
      "documentId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "lineNumber": 10000,
      "itemNumber": "1896-S",
      "description": "ATHENS Desk",
      "unitOfMeasureCode": "PCS",
      "quantity": 10
    }
  ]
}
```

> This endpoint is **read-only** (GET only). Receipts are created by posting purchase orders.

---

## OData Query Examples

### Filter by vendor

```
$filter=vendorNumber eq '10000'
```

### Filter by date

```
$filter=postingDate ge 2024-06-01 and postingDate le 2024-06-30
```

### Filter by order

```
$filter=orderNumber eq 'PO-1001'
```

---

## Common Errors

| Status | Error                  | Cause & Resolution                                                                 |
|--------|------------------------|------------------------------------------------------------------------------------|
| `401`  | Unauthorized           | Access token is missing, expired, or invalid.                                      |
| `404`  | Not Found              | The receipt does not exist. Verify the ID.                                         |
